---

### Overview

This project implements an end-to-end **speech-to-answer system**, combining **Automatic Speech Recognition (ASR)** and **Extractive Question Answering (QA)**.

* First, it converts spoken audio to text using ASR.
* Then, it allows the user to **ask questions about the transcribed text**, and answers them using a QA model.

---

### Features

*  Upload audio files or record live audio
*  View transcribed text
*  Ask questions on the transcript 
*  Get precise, extractive answers from your own speech content
*  Interactive web app (Streamlit)
